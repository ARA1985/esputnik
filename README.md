ESputnik
========

Marketing automation system ESputnik.

## curl
A basic CURL wrapper for PHP (see [http://php.net/curl](http://php.net/curl) for more information about the libcurl extension for PHP)

## Requirements
- **PHP** >= 5.4
- **EXT-CURL** = *
